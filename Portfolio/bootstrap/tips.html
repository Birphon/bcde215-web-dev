PuTTY Tips


Simply saving your PuTTY instance so you don't always need to enter in the same fields time and time again - You can even save a shortcut to your desktop
Note: On the Ara network if you go to a different PC this shortcut won't work - name the shortcut something along the lines of [RoomName]@[IPAddress]

You can also dupelicate the instance of PuTTY - This allows you to have multiplie connections to the same AWS Connection but have different things running i.e. MySQL, being in Apache2 Directory and Your Websites Directory

WinSCP Tips


Much like PuTTY, you can also save your WinSCP so you don't need to redo the fields. Again you can save to the desktop but you will run into the same issues on Ara Network

Bookmarks are a wonderful little thing. This will allow you to quickly hop around to different locations easily. I like to keep my /var/www/ directory, /etc/apache2/sites-avalible/ directory and the Error Logs directory bookmarked

You can also edit files within WinSCP and even use your favorite Text Editor - Note that saving files can be an issue if its Root since WinSCP doesn't have Root Permissions!
